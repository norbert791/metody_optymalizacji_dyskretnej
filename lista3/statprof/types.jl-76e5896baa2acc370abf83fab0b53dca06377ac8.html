3 """
4 AbstractInterpreter
5
6 An abstract base class that allows multiple dispatch to determine the method of
7 executing Julia code. The native Julia-LLVM pipeline is enabled by using the
8 `NativeInterpreter` concrete instantiation of this abstract class, others can be
9 swapped in as long as they follow the `AbstractInterpreter` API.
10
11 If `interp::NewInterpreter` is an `AbstractInterpreter`, it is expected to provide at least
12 the following methods to satisfy the `AbstractInterpreter` API requirement:
13 - `InferenceParams(interp::NewInterpreter)` - return an `InferenceParams` instance
14 - `OptimizationParams(interp::NewInterpreter)` - return an `OptimizationParams` instance
15 - `get_world_counter(interp::NewInterpreter)` - return the world age for this interpreter
16 - `get_inference_cache(interp::NewInterpreter)` - return the local inference cache
17 - `code_cache(interp::NewInterpreter)` - return the global inference cache
18 """

174 NativeInterpreter
175
176 This represents Julia's native type inference algorithm and the Julia-LLVM codegen backend.
177 It contains many parameters used by the compilation pipeline.
178 """
179 struct NativeInterpreter <: AbstractInterpreter
180 # Cache of inference results for this particular interpreter
181 cache::Vector{InferenceResult}
182 # The world age we're working inside of
183 world::UInt
184 # method table to lookup for during inference on this world age
185 method_table::CachedMethodTable{InternalMethodTable}
186
187 # Parameters for inference and optimization
188 inf_params::InferenceParams
189 opt_params::OptimizationParams
190
191 function NativeInterpreter(world::UInt = get_world_counter();
192 inf_params = InferenceParams(),
193 opt_params = OptimizationParams(),
194 )
195 cache = Vector{InferenceResult}() # Initially empty cache
196
197 # Sometimes the caller is lazy and passes typemax(UInt).
198 # we cap it to the current world age
199 if world == typemax(UInt)
200 world = get_world_counter()
201 end
202
203 method_table = CachedMethodTable(InternalMethodTable(world))
204
205 # If they didn't pass typemax(UInt) but passed something more subtly
206 # incorrect, fail out loudly.
207 @assert world <= get_world_counter()
208
209 return new(cache, world, method_table, inf_params, opt_params)
210 end
211 end

230 """
231 lock_mi_inference(::AbstractInterpreter, mi::MethodInstance)
232
233 Hint that `mi` is in inference to help accelerate bootstrapping.
234 This is particularly used by `NativeInterpreter` and helps us limit the amount of wasted
235 work we might do when inference is working on initially inferring itself by letting us
236 detect when inference is already in progress and not running a second copy on it.
237 This creates a data-race, but the entry point into this code from C (`jl_type_infer`)
238 already includes detection and restriction on recursion, so it is hopefully mostly a
239 benign problem, since it should really only happen during the first phase of bootstrapping
240 that we encounter this flag.
241 """
242 lock_mi_inference(::NativeInterpreter, mi::MethodInstance) = (mi.inInference = true; nothing)
243 lock_mi_inference(::AbstractInterpreter, ::MethodInstance) = return
